# VitraView
## Overview

A novel approach for window-like experiences. 


## Components
- Camera intrinsics utility.
- Novel camera extrinsics to monitor technique.
- 3D Eye tracking from IPD and pixel coordinates.
- Kalman filtering.
- Unity demo project.
